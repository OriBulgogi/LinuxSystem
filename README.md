# LinuxSystem
Linux os , c 시스템 프로그래밍


#### mycp.c
- 셸 명령어인 cp 구현
#### myls.c
- 셸 명령어인 ls 구현
#### mycat.c
- 셸 명령어 cat 구현(option: n, E)
#### myshell.c
- cd, pwd, exit, help 를 내장 명령으로 가지는 shell 프로그램 구현
#### thread_link.c
- 여러 개의 쓰레드가 하나의 자료 구조(LinkedList)를 공유하는 프로그램 작성
